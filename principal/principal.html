Para ti

Este tarro guarda un pedacito de mí, para cada momento en el que me necesites yo pueda estar para tí.

Cuando estés feliz, para verte sonreír aún más.

Cuando estés triste, para recordarte que no estás sola.

Cuando estés enojada, para darte calma.

Cuando estés desmotivada, para levantarte esos ánimos

Cuando estés preocupada, para abrazarte aunque sea de esta forma.

Todo esto lo hice pensando en ti

Porque me importas más de lo que puedo demostrar

Así que cuando no sepas qué hacer, o solo quieras sentirte mejor, elige una nota.

Siempre habrá algo aquí para ti, mi linda princesita ❤️

Rosa = cuando estés feliz
Celeste = cuando estés triste
Rojo = cuando estés enojada
Naranja = cuando estés desmotivada
Verde = cuando estés preocupada
